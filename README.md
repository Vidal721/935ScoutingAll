# 935/757 Scouting System

## What does this include
- Scouting app for any android tablet
- A scanner and scouting dashboard
- And a HTML page you can use to host your website

## What you need
- 6 scouting tablets
- 1 scouting laptop
- 1 leadscout tablet (optional)
- Webhosting Platform (We use ([netlify])https://netlify.com)
